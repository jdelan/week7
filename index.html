JavaScript Cheat Sheet
  1. Wondering about a variable's value? Make it visible by using the console: 
    
console.debug("My variable 'obj' has the value: ")
console.debug(obj)
  2. Wondering about a variable's type? 
    
console.debug("The variable 'obj' is of type: ")
console.debug(typeof(obj))
  3. Trying to access a specific element of an array? 
    
let names = ["Cookie Monster", "Big Bird", "Elmo"];
let bird = names[1];
  4. Trying to access a specific value of an object? 
    
let person = { name: "Cookie Monster", favoriteFruit: 'Cookies' };
let snack = person.favoriteFruit;
// or
let snack = person["favoriteFruit"]
